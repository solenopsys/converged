# JSON Engine (example)

Назначение
- пример файлового JSON-хранилища для простых сущностей.

Backend
- файловая система.

Ключи и данные
- ключ — список сегментов пути (см. `example.ts`).
- значение сериализуется в JSON.

API (пример)
- `get`, `set`, `delete`, `listKeys`

Миграции
- не используются (примерный код).
