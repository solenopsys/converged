# Cache dir 
 - importconfig - сохраняет конфиг импортов которые используются на сервере для динамической генерации configmap
 - importmaps - сохраняет importmap файлы для каждого пакета
 - libraries - хранит скомпилированные библиотеки
 - sourcemaps - хранит соответствующие карты исходников
 - temp - хранит временные файлы это используется Bun.build
 - meat.json - хранит метаданные для кеширования и поиска библиотек
